# AEJ: Applied submission materials

Submission preparation for *"When Quality (doesn't) Pay: Evidence from Two Experiments
in Ugandan Dairy"* (Van Campenhout, Ariong, Kariuki, Chamberlin) to the American
Economic Journal: Applied Economics.

Folder is named `AEJ_applied` (not `AEJ:applied`) because colons are not portable
across macOS Finder and Windows filesystems.

## Disclosure statements (`disclosure_statements/`)

One pre-filled DOCX per co-author, following the AEA Disclosure Policy (five numbered
items: funding sources; support from interested parties >= USD 10,000 over three
years, including in-kind; relevant positions; close relatives and partners; right of
review). Item 1 is pre-filled with the CGIAR Rethinking Markets Research Initiative
and the CGIAR Better Diets and Nutrition Science Program; items 2-5 are pre-filled
with nothing-to-disclose language.

Workflow: each co-author edits any statement that does not apply, deletes the
bracketed instruction paragraph, signs, dates, and returns a PDF named
`Disclosure_Statement_<Name>.pdf`. The AEA requires a separate PDF per author,
submitted with the manuscript and visible to referees.
